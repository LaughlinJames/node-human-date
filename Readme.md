# node-human-date

A node.js package for formatting a date in Twitter's human readable style.

## Sample return values

15s, 6m, 18h, 4d, 7 Sep


## Usage
## code
parseDate = 'Sun Sep 09 2012 22:43:08 GMT-0500 (CDT)'
var humanDate = human.formatDate( parseDate );
